Documentaries

Documentaries es un proyecto creado para ser un buscador de documentales.

Este proyecto se crea por la necesidad de practicar WEB SCRAPING y DJANGO. Los documentales son extraídos de las siguientes páginas:

  • ‣ Documentary Mania
  • ‣ DocumentaryTube
  • ‣ Documentary Addict
  • ‣ Documentary Heaven
  • ‣ Top Documentary Films

El código del scraping y el de este sitio están en gitlab y github.
